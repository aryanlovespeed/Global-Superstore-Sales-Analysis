# 📊 Global Superstore Sales Analysis

## Project Overview

This project analyzes the Global Superstore dataset using Microsoft Excel to uncover sales trends, customer behavior, product performance, and regional business insights. The dashboard was built to help business stakeholders identify opportunities for improving sales and profitability.

---

## Objective

The objective of this project was to transform raw transactional data into meaningful business insights through data cleaning, analysis, and an interactive dashboard.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Lookup Functions (XLOOKUP / INDEX-MATCH)
- Excel Formulas
- Dashboard Design

---

## Data Preparation

The dataset was cleaned before analysis by:

- Checking and removing duplicate records
- Handling missing values
- Correcting inconsistent formatting
- Creating helper columns where required
- Preparing data for Pivot Tables

---

## Key KPIs

- Total Sales
- Total Profit
- Profit Margin %
- Total Orders
- Average Delivery Days
- Total Customers

---

## Business Analysis

The dashboard provides insights into:

- Sales performance by Year and Month
- Regional Sales Comparison
- Country, State and City Performance
- Category & Sub-Category Performance
- Customer Sales Analysis
- Profitability Analysis
- Shipping Performance
- Top Performing Products
